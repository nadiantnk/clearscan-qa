## О проекте

ClearScan — однофайловое SPA-приложение (HTML + CSS + JS) для записи пациентов в диагностический центр КЛКТ/ОПТГ.

Что умеет приложение:
- расписание с шагом 15 минут (9:00–20:00)
- создание, редактирование и удаление карточек пациентов
- валидация полей формы (ФИО, телефон, email, скидка)
- статусы визита — пришёл / не пришёл, подсветка строки
- выбор способа оплаты (карта, наличные, перевод)
- финансовый отчёт с фильтрами и суммами по видам оплаты
- справочники: клиники, врачи, услуги с ценами
- настройки: управление филиалами, рабочее время
- данные хранятся в localStorage

**Стек:** HTML5, CSS3, Vanilla JS, localStorage  
**Браузеры:** Chrome 123+, Firefox 124+, Edge 123+
